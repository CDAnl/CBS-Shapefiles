CBS-Shapefiles
==============

Dit is een repository met alle CBS-shapefiles in GeoJSON-formaat.  
De shapefiles zijn geprojecteerd op EPSG:4326 en dus meteen bruikbaar in [Leaflet](https://github.com/Leaflet/Leaflet).
